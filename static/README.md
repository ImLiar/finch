The `finch-json4s` module provides the Finch library a support for serving static files via HTTP.

Installation
------------
Use the following _sbt_ snippet:

```scala
libraryDependencies ++= Seq(
  "com.github.finagle" %% "finch-static" % "0.8.0"
)
```
